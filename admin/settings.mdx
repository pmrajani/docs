Admins manage workspace configuration from **Settings** in the sidebar. The settings area is divided into several sections.

## Company & documents

Go to **Settings → Company** to update your company profile. This information appears on quotation and invoice PDFs sent to customers.

| Field | Notes |
|---|---|
| **Company name** | Printed in the seller block of all PDFs |
| **Address** | Full mailing address for the seller block |
| **Phone** | Contact number shown on PDFs |
| **Email** | Business email shown on PDFs |
| **GSTIN** | Your GST Identification Number — enables tax calculation on invoices |
| **Website** | Optional; printed on PDFs |
| **State name / State code** | Used to calculate IGST vs CGST/SGST split |

### PDF template

Choose the invoice layout WapNotify uses for all quotations and sales orders:

| Template | Description |
|---|---|
| **Modern** | Clean, minimal design |
| **Tally GST Tax Invoice** | Bordered Tally-style layout with full GST breakdown — HSN summary, CGST/SGST or IGST columns, authorised signatory block |

Click **Save** after making changes.

## WhatsApp setup

Go to **Settings → WhatsApp Setup** to manage your WhatsApp Business API connection — credentials, webhook URL, and phone number info.

See [WhatsApp setup](/admin/whatsapp-setup) for the full guide.

## Team

Go to **Settings → Team** to:
- View all workspace members
- Invite new members
- Change a member's role
- Deactivate members

See [invite your team](/getting-started/invite-team).

## Roles

Go to **Settings → Roles** to create and manage custom roles with granular permissions.

See [roles & permissions](/admin/roles-permissions).

## Tags

Go to **Settings → Tags** to create and manage the tag catalog used for labeling contacts.

Tags must exist in the catalog before they can be assigned to contacts. Deleting a tag from the catalog removes it from all contacts.

## Quick replies

Go to **Settings → Quick Replies** to manage the library of saved message snippets available to agents.

Each quick reply has a **title** (used for search in the `/` picker) and a **message** body.

## Custom fields

Go to **Settings → Custom Fields** to define additional fields for contact records.

See [custom fields](/contacts/custom-fields).

## Modules

Go to **Settings → Modules** to toggle optional feature modules on or off for your workspace:

| Module | What it enables |
|---|---|
| **Leads** | The Leads Kanban pipeline |
| **Store** | Quotations, Sales Orders, and the Product catalog |
| **Products** | Product catalog (within Store) |

Disabling a module hides it from the sidebar. Existing data is preserved — re-enabling the module restores access to it.

### Conversation status workflow

The **Conversations** card in Modules includes a sub-toggle for the status workflow. When enabled, conversations have Open / Pending / Resolved statuses and agents can filter the inbox by status. Disable it to hide status tabs and treat all conversations as a single flat list.

## Data management

Go to **Settings → Danger zone** for irreversible workspace actions.

**Delete all workspace data** — permanently deletes all contacts, conversations, messages, broadcasts, templates, leads, and settings for the workspace. The workspace account itself is not deleted, but it is reset to a clean state.

<Warning>
  Deleting workspace data is permanent and cannot be undone. Export any data you need before proceeding.
</Warning>
